# java-cli-buildr-hello-world

## Description
This is a POC project to demonstrate the
apache buildr build tool.

## Tech stack
- buildr
- java

## Docker stack
- vanto/apache-buildr:latest-jruby-jdk8

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
